# Daniel Iradukunda — Portfolio

A modern and interactive developer portfolio built with React, TypeScript, Tailwind CSS, and Framer Motion.

This portfolio showcases my projects, experience, design approach, blogs, and creative journey as a Front-End Developer and UI Designer focused on building scalable and visually refined digital experiences.

---

## ✨ Features

- Modern responsive UI
- Smooth animations with Framer Motion
- Interactive project showcase
- Blog section with modal previews
- Animated typing hero section
- Experience timeline
- Contact form integration
- Mobile-friendly design
- Clean reusable component structure
